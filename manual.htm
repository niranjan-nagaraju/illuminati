USER MANUAL  

 

 

 

                Illuminati – the graffiti canvas is a 2D graphics editor that seeks to provide an user friendly interface to enable users to create , edit , load and save images.

 

 

 

Ù     The Menubar

                Provides different menus that can be used to perform common operations.

The menu break up and their functions are as listed below.

 

 

      

File

provides file operations

 

 

New

Opens a new file

 

Open

Opens an existing file

 

Save

Saves current image to a file

 

Save As

Saves current image to a specified file

 

Print

Prints the image

 

Exit

Quits the application

 

 

 

 

Edit

provides editing operations

 

 

Undo

Undoes previous command

 

Redo

Redoes previous command

 

Cut

Cuts a selection

 

Copy

Copies a selection

 

Paste

Pastes a cut or copied selection

 

Clear

Clears a selection

 

Select All

Selects the entire viewport

Ù    The Drawing area                                                         

 

Which is a WYSYWIG interface that reflects the result of various operations performed on the images.

       The drawing area maps the image into a 640 x 480 bitmap for easier editing.

 

 

 

Ù    The Dialog Boxes

 

 

Several dialog boxes are used to provide user interaction and accept user input.

    The various dialog boxes used are :

 

Y      File selection dialog        

                                To accept filename(s) to open / save images.

 

Y      Color selection dialog

                                To provide a WYSYWIG interface that enables users to choose various colors.

 

 

Y      Transformation dialogs

                       That accepts various transformation parameters such as rotation angle , translation displacement ,

etc.

 

Y      Font control dialog

                        That enables users to choose between various fonts  , sizes and  styles.

 

 

Y      File save/discard dialog

                          That alerts the user that s/he failed to save the current file and has attempted to quit the

application.

 

 

 

Ù     The Toolbar

 

Provides an iconic interface to choose various tools 

 

v     Pencil  for freehand drawing.

v     Selector

v     Bucket filler

v     Color chooser

v     A tool each for drawing primitives and

v     A tool for handling text input.

 

The tooltips associated with tool icons provides context sensitive help.

 

 

 

Ù     The Color Box

 

Provides a history of recently used and commonly used colors.

 

 

Ù     The Status Bar

 

Provides context sensitive help messages to aid novice users.

 

 

Ù     A Top Level Window

 

Which houses all these components and provides common window operations such as close , minimize , maximize and resize , etc.
